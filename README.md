# 꼽주머니 (Kkop-jumoney)

> **"팩폭이 지출을 막는다."**
> 돈 모을 때까지 욕먹는 AI 가계부 프로젝트

## 📖 프로젝트 소개 (About Project)

**꼽주머니**는 익명의 다수가 불필요한 소비를 비판해주는 '카카오톡 거지방'에서 영감을 받아 탄생했습니다[3]. 
기존 가계부 앱들이 부드러운 조언을 건넨다면, 꼽주머니는 사용자의 지출 내역을 분석해 뼈를 때리는 '팩트 폭력'과 '병맛 알림'으로 강력한 동기부여를 제공합니다[4].

## 핵심 기능 (Key Features)

### 1. 팩트 폭력 알림 (Roasting Notification)
* **지출 패턴 감시:** 예산 초과 후 커피 구매, 저축 목표 실패 등 10가지 이상의 트리거를 감지합니다[4].
* **꼽주는 메시지:** "돈 냄새 나니?", "그 돈이면 국밥이 한 그릇" 등 AI가 생성한 강도 높은 꾸중 메시지를 발송합니다.
* **강도 조절:** 순한맛 / 매운맛 / 팩폭맛 중 사용자가 원하는 욕설 강도를 설정할 수 있습니다[6].

### 2. 반응형 거지 캐릭터 (Reactive Character)
* **감정적 UI:** 사용자의 월 소비액과 예산 달성률에 따라 메인 대시보드의 캐릭터 상태가 실시간으로 변합니다[4, 7].
* **시각적 피드백:** 
    * 지출 적음: 스포츠카를 탄 여유로운 상태
    * 지출 보통: 벤치에 앉은 평범한 상태
    * 지출 과다: 비를 맞으며 처량한 상태
    * 파산 직전: 깡통 차고 구걸하는 상태 [8]

### 3. AI 비난 리포트 & 자동화
* **자동 기록:** Plaid API를 연동하여 은행 및 카드 거래 내역을 자동으로 수집하고 분류합니다[4, 9].
* **월간 리포트:** 월말에 가장 비합리적이었던 소비 습관을 분석하여 "당신의 낭비 역사가 업데이트되었습니다"라는 식의 비난 리포트를 발행합니다[4, 10].
